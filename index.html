Two apps,
one schedule

GymOrgPro holds the timetable — squads, blocks, rotations, who’s coaching what. Chalk turns any one of those sessions into a lesson plan you can print or hand to a coach. They read the same database, so the schedule is only ever entered once.

The schedule

GymOrgPro

Build the term. Set up squads, blocks and the rotation grid, then work through the calendar week by week.

  • Squads, coaches, session times and lesson-plan banners
  • Rotation grids — which station, for how long, in what order
  • Warm-up, warm-down and standing notes for every plan
  • Word lesson plans in bulk, for a squad or a whole block
Open GymOrgPro →

The lesson

Chalk

Fill in the session. Pick a lesson from the schedule and tick the skills into each rotation.

  • Thousands of club skills with coaching points and diagrams
  • Warm-up and warm-down activities alongside the apparatus
  • The MAG ALP pathway, filtered to a squad’s working level
  • Edit any skill, add your own, map one to several apparatus
Open Chalk →

They’re linked. In GymOrgPro’s calendar, every session has a “Chalk ↗” button that opens Chalk on that exact lesson — right squad, right date, real rotation, nothing to re-enter. Chalk reads the schedule but can never change it, so GymOrgPro stays the single source of truth.

Both apps run in the browser with no install. Keep this folder together and open index.html, or serve the folder from any web host. See README.md for how the pieces fit together and how to rebuild Chalk after editing it.
