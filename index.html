Rotation Settings ADMIN ONLY

How long each rotation slot lasts. All session durations round to the nearest multiple of this value. Existing station picks carry over automatically: shrinking this value is lossless (each pick fills the new, smaller slots it spans), while growing it combines multiple slots into one and keeps only the earliest pick where they conflict.

Team & Access ADMIN ONLY

Add everyone who needs to open this schedule and set what they can do — the principle is to give each person the least access that still lets them do their job. Admin: full control, including this panel, Program Details and Rotation Settings. Scheduler: day-to-day editing — Training Schedule, Rotations, Staffing, plus Squads/Stations/Coaches/Conflicts/Capacity — but not the settings above. Coach: can view every tab, including their own roster, but can't change anything. Viewer: read-only — good for a front-desk screen or a link shared with parents.

Each person signs in once per browser with their name and PIN (below the shared password). Like the shared password, this is a lightweight, prototype-level lock — the PINs live in this file's data, not a real authentication system. For a production rollout, swap this for proper authentication (e.g. Firebase Authentication) tied to the same roles.

Generate Lesson Plans

Produces one Word document per session in the date range below, all named clearly (squad, term, week, date, day, time) and packaged flat in a single zip — no subfolders to dig through. Each document's header, coach details, and circuit list (station + duration) are filled in automatically from Training Schedule and Rotations, styled to match the club's own lesson plan template. The Skill/KCP/Safety and warm-up/warm-down details are left blank for coaches to fill in themselves.

The Rotations tab's two-week cycle repeats indefinitely — this tells the generator which real Monday lines up with "Week 1" so it picks the correct station allocations for each date. It defaults to the Monday on/before "From date".

Type any label freely, or pick one from the dropdown that matches an uploaded header (see Setup → Lesson Plan Headers) to use that banner instead of the default.
